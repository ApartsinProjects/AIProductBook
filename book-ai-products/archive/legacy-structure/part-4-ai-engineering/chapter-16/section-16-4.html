Big Picture: Security Is Not Optional

This section covers security and compliance considerations for AI systems including API security, data privacy, and compliance frameworks like SOC2, HIPAA, and GDPR. In the context of AI engineering, security failures can be existential risks. Compliance violations can halt product launches and damage reputation.

Fun Note: The API Key That cost $2 Million

A developer accidentally committed an API key to a public GitHub repository. Within 24 hours, someone had found it and run up $2 million in charges. The fix took 5 minutes. The bill took months to negotiate. Always use environment variables and secret managers, never commit keys.

16.4 Security and Compliance

AI systems face unique security and compliance challenges. Beyond traditional application security, you must protect against prompt injection, data leakage through outputs, and model-specific vulnerabilities. Compliance requirements like SOC2, HIPAA, and GDPR add additional obligations.

Why This Matters

Security failures in AI systems can expose sensitive data, allow unauthorized access, or enable harmful outputs. Compliance violations can halt product launches, result in fines, and damage reputation. Both are existential risks for AI products.

1. API Security

Secure your AI APIs with proper authentication, authorization, and rate limiting. This is the first line of defense against abuse.

Practical Example: API Security Implementation
class SecureAIAPI:
    """Secure API implementation with multiple layers"""
    
    def __init__(self):
        self.auth_provider = Auth0Client()
        self.rate_limiter = TokenBucketLimiter()
        self.api_key_manager = APIKeyManager()
    
    async def handle_request(self, request: Request) -> Response:
        # Layer 1: Authentication
        user = await self.auth_provider.authenticate(request.headers)
        if not user:
            return Response(status=401, error="Unauthorized")
        
        # Layer 2: Rate limiting
        if not self.rate_limiter.check(user.id):
            return Response(status=429, error="Rate limit exceeded")
        
        # Layer 3: Authorization
        if not self.can_access(user, request.endpoint):
            return Response(status=403, error="Forbidden")
        
        # Layer 4: Process request
        result = await self.process(request, user)
        
        # Layer 5: Audit logging
        await self.audit_log.log(user.id, request, result)
        
        return result
    
    def can_access(self, user: User, endpoint: str) -> bool:
        # Check user's permissions
        permissions = user.get_permissions()
        return endpoint in permissions

# API key best practices:
# - Rotate keys regularly (90 days)
# - Use scopes to limit capabilities
# - Monitor usage patterns
# - Revoke compromised keys immediately

2. Data Privacy

AI systems often process sensitive data. Implement proper PII handling, encryption, and data minimization to protect user privacy.

Case Study: PII Handling
class PrivacyProtector:
    """Protect privacy throughout the AI pipeline"""
    
    def __init__(self):
        self.pii_detector = load_pii_detector()
        self.encryption = AESEncryption()
    
    async def process(self, user_input: str, user_id: str) -> ProcessedInput:
        # Detect PII in input
        pii_locations = await self.pii_detector.detect(user_input)
        
        # If PII found, either block or anonymize based on policy
        if pii_locations and not self.user_consents_to_pii(user_id):
            return ProcessedInput(
                allowed=False,
                reason="Contains detected PII - please remove personal information"
            )
        
        # Anonymize PII for processing
        anonymized = self.anonymize(user_input, pii_locations)
        
        # Store only anonymized data for improvement
        await self.store_anonymized(anonymized)
        
        return ProcessedInput(text=anonymized, contains_pii=bool(pii_locations))
    
    def anonymize(self, text: str, pii_locations: list) -> str:
        for pii in pii_locations:
            if pii.type == "email":
                text = text.replace(pii.text, f"user_{hash(pii.text)}@redacted")
            elif pii.type == "phone":
                text = text.replace(pii.text, "REDACTED")
            # ... other types
        return text
    
    # Encrypt data at rest and in transit
    def encrypt_for_storage(self, data: str) -> str:
        return self.encryption.encrypt(data)
    
    def decrypt_for_use(self, encrypted_data: str) -> str:
        return self.encryption.decrypt(encrypted_data)

4. Audit Trails

Complete logging enables security audits, incident investigation, and compliance demonstration. Every significant action should be logged.

Skill Demonstration: Comprehensive Audit Logging
class AuditLogger:
    """Complete audit logging for compliance"""
    
    async def log(self, event: AuditEvent):
        entry = {
            "timestamp": datetime.now().isoformat(),
            "event_type": event.type,
            "user_id": event.user_id,
            "ip_address": event.ip_address,
            "action": event.action,
            "resource": event.resource,
            "result": event.result,
            "metadata": event.metadata,
            # Hash for integrity verification
            "content_hash": hash(event.serialize())
        }
        
        # Write to tamper-proof storage
        await self.secure_store.write(entry)
    
    def generate_audit_report(
        self, 
        start_date: datetime, 
        end_date: datetime,
        user_id: str = None
    ) -> AuditReport:
        # For compliance audits
        entries = self.secure_store.query(
            start=start_date,
            end=end_date,
            user=user_id
        )
        
        return AuditReport(
            entries=entries,
            summary=generate_summary(entries),
            certificate=self.generate_integrity_certificate(entries)
        )

# What to log:
# - Authentication (login, logout, failures)

# - Security events (suspected attacks)
